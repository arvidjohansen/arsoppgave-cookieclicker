Hvordan sette opp Button Masher (TL;DR edition):

1. Last ned filer fra GitHub og legg de i htdocs-mappa på webserveren (www om du bruker noe annet enn XAMPP)
2. Åpne eventuell brannmur
3. Kjør scriptene i sql-katalogen for å lage database, tabeller, db-bruker og testdata
4. Endre IP-adresse (og eventuelt db-bruker) i connection_info.php
5. Skriv inn IP-adressen til serveren i klient-browseren, og sjekk at alt funker

For mer detaljert info, se instruksjoner i katalogen 'Teknisk dokumentasjon'
